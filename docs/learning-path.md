## ðŸ’» Running Go Files

To run a Go file, use the following command:

```bash
go run filename.go
```

For example:
- `go run 01-variables/main.go`
- `go run 02-fmt/main.go`
- `go run 03-booleans/main.go`

You can also run multiple files at once:
```bash
go run file1.go file2.go
```
